# Student Portfolio Builder Skill

This skill provides the knowledge and patterns required to build a creative and professional student portfolio website.

## Core Capabilities
- **Spring Boot Routing**: Implementing clean controllers for multi-page static sites.
- **Thymeleaf Templating**: Creating dynamic yet performant HTML layouts.
- **Modern UI/UX**: Designing responsive and visually appealing interfaces using Vanilla CSS.

## Guidelines
1. **Storytelling**: Ensure the portfolio tells a cohesive story about the student's journey.
2. **Visual Impact**: Use high-quality images and subtle animations (hover, fade-in).
3. **Consistency**: Maintain a unified design system throughout all pages.

## Project Structure
- `src/main/resources/templates/`: Thymeleaf HTML files.
- `src/main/resources/static/css/`: Vanilla CSS files.
- `src/main/resources/static/images/`: Portfolio images.
