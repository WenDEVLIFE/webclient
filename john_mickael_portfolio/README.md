# My Student Life Portfolio Website

A personal portfolio website built with **Spring Boot** and **Thymeleaf**, showcasing my journey as a BSIT student.

## 🚀 Technologies Used
- **Language**: Java 21
- **Framework**: Spring Boot 4.0.5
- **Template Engine**: Thymeleaf
- **Styling**: Vanilla CSS (Modern, Responsive, Animated)
- **Design**: Premium glassmorphism effects and custom animations

## 📂 Project Structure
- `src/main/java/trist/portfolio/`: Backend controllers and application logic.
- `src/main/resources/templates/`: Thymeleaf HTML templates (Home, About, Journey, Skills, Gallery).
- `src/main/resources/static/css/`: Custom CSS for all pages.
- `src/main/resources/static/images/`: Images representing my student life.

## 📄 Pages
1. **Home**: Quick introduction and welcome message.
2. **About Me**: Detailed biography, interests, and student goals.
3. **My College Journey**: Key highlights and challenges in my IT studies.
4. **Skills and Interests**: Technical skills overview and personal hobbies.
5. **Photo Gallery**: Visual diary of my college experiences.

## 🛠️ How to Run
1. Clone the repository.
2. Ensure you have Java 21 installed.
3. Run the application using Maven:
   ```bash
   ./mvnw spring-boot:run
   ```
4. Access the website at `http://localhost:8080`.
