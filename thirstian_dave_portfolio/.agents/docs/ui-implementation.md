# UI Implementation Plan

## Design System
- **Theme**: Modern, clean, and "vibrant" or "premium" as per development guidelines.
- **Typography**: Google Fonts (e.g., 'Inter' or 'Poppins').
- **Colors**: A professional palette (e.g., Deep Blues, Soft Grays, or a personalized student theme).
- **Navbar**: Sticky navigation with links to Home, About, Journey, Skills, Gallery.

## Page Layouts
- **Home**: Hero section with profile photo and welcome message.
- **About**: Two-column layout for biography and interests.
- **Journey**: Timeline or card-based layout for college experiences.
- **Skills**: Grid of icons and progress bars/tags for programming skills.
- **Gallery**: Responsive masonry or grid layout for photos with hover captions.

## Interactions
- Hover effects on navigation links and gallery images.
- Smooth transitions between sections/pages.
- Responsive mobile menu.
